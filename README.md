## Development

- Install [Ruby][ruby]
- Install [bundler][bundler]
- Install [Jekyll][jekyll]
- Install [nokogiri][nokogiri]
- Clone the repo, `cd` into repo
- `bundle install`
- `bundle exec jekyll serve`
